SLONIK EXIT

Name

EXIT --  Terminate Slonik script with signal 返り値を指定してslonikスクリプトを終了

Synopsis 使用方法

exit [ [-]ival]

Description 解説

Terminates script execution immediately, rolling back every open transaction on all database connections. The slonik utility will return the given value as its program termination code. slonikスクリプトの実行を直ちに停止し、全てのデータベース接続に対する全ての実行中のトランザクションをロールバックします。 slonik ユーティリティは与えられた値をプログラム停止コードとして呼び出し元に返します。

Example 例

     EXIT 0;

Version Information バージョン情報

This command was introduced in Slony-I 1.0 このコマンドは Slony-I 1.0 で導入されました。
