SLONIK INCLUDE

Name 名称

INCLUDE --  pulling in slonik code from another file slonikスクリプトを他のファイルから引用する

Synopsis 使用方法

include [ <pathname>]

Description 説明

This draws the specified slonik script inline into the present script. If thepathname specifies a relative path, slonik will search relative to the current working directory. これは指定されたslonikスクリプトを、現在のスクリプトに取り込むコマンドです。もしも pathname が相対パスであるならば、 slonikは現在の作業ディレクトリを起点としてファイルを探します。

Nested include files are supported. The scanner and parser report the proper file names and line numbers when they run into an error. 入れ子（includeされるファイル内でのincludeコマンド）もサポートされています。エラーが発生した場合、スキャナとパーサは正しいファイル名と行数を報告します。

Example 例

     include </tmp/preamble.slonik>;

Version Information バージョン情報

This command was introduced in Slony-I 1.1 このコマンドは Slony-I 1.1 で導入されました。
