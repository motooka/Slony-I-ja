1.108. slon_quote_brute( text )

Function Properties

Language: PLPGSQL

Return Type: text

Brutally quote the given text 

declare	
    p_tab_fqname alias for $1;
    v_fqname text default '';
begin
    v_fqname := '"' || replace(p_tab_fqname,'"','""') || '"';
    return v_fqname;
end;
