SLONIK SLEEP

Name

SLEEP --  Sleep using system sleep()

Synopsis

sleep [ seconds]

Description

Sleeps for the specified number of seconds.

Example

     sleep (seconds = 5);

Version Information

This command was introduced in Slony-I 1.1.6 / 1.2.1.
